# Лабораторная работа №3
Тема: Проектирование и оптимизация архитектур нейронных сетей для задач классификации изображений.

## Задачи

### 1. Классификация рукописных цифр MNIST (Keras)
Разработали сверточную нейронную сеть для распознавания рукописных чисел с соблюдением заданных ограничений по архитектуре.
- Архитектура: **минимум 2 сверточных слоя**, слои подвыборки, нормализации и регуляризации (**BatchNormalization** + **Dropout**).
- Параметры обучения: **15 эпох**, **batch_size=128**, оптимизатор **Adam (lr=0.001)**, loss - **categorical_crossentropy**.
- **Результат:** сформированы графики доли верных ответов для обучающей и проверочной выборок.

### 2. Классификация пассажиров автобуса (CNN, Keras)
Реализовали нейронную сеть для бинарной классификации пассажиров на «входящих» и «выходящих».
- Задача: достижение точности работы модели **выше 90%** на проверочной выборке.
- Особенности: оптимизация архитектуры сверточной сети для работы с видеокадрами.
- **Результат:** целевой порог точности в **90%** успешно преодолен.

### 3. Классификация базы автомобилей (Transfer Learning)
Создали систему распознавания марок автомобилей с использованием различных подходов к обучению и аугментации.
- Методы: использование **ImageDataGenerator**, кастомных генераторов и готовых архитектур из **tf.keras.applications**.
- Особенности: самостоятельное изучение и внедрение предобученных моделей для классификации.
- **Результат:** точность распознавания на проверочной выборке составила **не ниже 93%**.
